Project Value Noise

3D Graphics

Project Overview

Implemented a Value Noise system capable of generating smooth procedural noise textures through interpolation and configurable smoothing methods.

Tasks Completed

  • Implemented the core components of a Value Noise pipeline, including coordinate processing, permutation hashing, and interpolation between sampled values
  • Developed support for multiple smoothing methods (Linear, Cosine, Smoothstep, and Quintic) and applied them during noise evaluation.

Reflection

Through this project, I gained a better understanding of Value Noise and how interpolation and smoothing functions affect the final result. I also became more comfortable implementing mathematical systems from a specification.
